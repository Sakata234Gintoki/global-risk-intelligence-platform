# Global Risk Intelligence Platform

## Status: Month 1 — Data Engineering Core (In Progress)

## What This Project Does
A production-grade platform that automatically collects global health,
finance, and climate data, cleans it, stores it in a structured warehouse,
and uses ML to detect risk patterns across countries.

## Stack
- PostgreSQL (data warehouse — star schema)
- Apache Airflow (pipeline orchestration)
- AWS S3 (data lake)
- PySpark (batch processing)
- dbt (data transformation)
- Docker (containerization)
- GitHub Actions (CI/CD)
